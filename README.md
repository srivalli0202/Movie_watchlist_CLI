# Movie_watchlist_CLI
## Description
Movie Watchlist CLI is Ruby -based command-line application.It helps track movies they want to watch, mark watched movies and reviews. It is simple tool for movie lovers to organize their watchlist and remember of each film.

## Features
>1 Add movie
>2 Mark movie as watched
>3 Enter Add review
>4 Enter view watchlist
>5 Enter Exit
- View all movies with status and details

 ## Usage
In the terminal you need to Run ruby app.rb
