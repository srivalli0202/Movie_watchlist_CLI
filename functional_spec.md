# Movie Watchlist CLI

The Movie Watchlist CLI app provides terminal based Ruby application that helps user manage a personal movie watchlist. It allows user to add movies, mark them as watched, reviews and ratings.

## User Stories
. As a user, I want to mark movies as watched so I can trak my movies.
. As a reviewer, I want to rate and review movies.
. As a movie lover, I want to add movies to my watchlist so I can check.

## Functional Requirements
. User can add movie by title
. User can add rating and review to watched movies
. User can view all movies with detail
. User can mark a movie as watched
. User input is handled via'gets.chomp'
. Classes use 'initialize', variables, and accessors
